This is CURRENCY CONVERTER Project

Currency Converter:
The Currency Converter is a beginner-friendly application designed to convert a given amount from one currency to another using predefined or API-based exchange rates. Users simply enter the amount and choose the currencies, and the program instantly calculates the converted value. This project demonstrates basic mathematical operations, input handling, and function usage in programming. It is simple, efficient, and can be easily extended by adding real-time exchange rates or additional currency options, making it a great practical tool for learning and experiments.
